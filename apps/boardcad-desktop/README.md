# Ripple Studio (web app)

React + Vite front-end for the surfboard `.brd` editor.

## From this directory

```bash
npm install
npm run dev
npm run typecheck
npm run check:no-tauri
npm run test
npm run build
```

The workspace package name is **`boardcad-web`**; run scripts via the repository root with `npm run <script> -w boardcad-web` if needed.
