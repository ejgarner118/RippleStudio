# Ripple Studio (desktop)

Tauri + React front-end for the surfboard `.brd` editor.

## From this directory

```bash
npm install
npm run dev          # Vite only
npm run tauri:dev    # Tauri + Vite (see repo root for Windows MSVC notes)
npm run typecheck
npm run test
npm run build
```

The workspace package name is **`ripple-desktop`**; run scripts via the repository root with `npm run <script> -w ripple-desktop` if needed.
